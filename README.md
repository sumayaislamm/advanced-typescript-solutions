# TypeScript Advanced Problem Solving & OOP Assignment

## Project Overview
This project contains solutions to a TypeScript assignment focused on fundamental and advanced concepts such as:

- Functions
- Array manipulation
- TypeScript types and type guards
- Union types
- Interfaces
- Generics (basic usage)
- Object-Oriented Programming (OOP)
- Inheritance and classes

All solutions are implemented in a single file: `solutions.ts`.

## How to Run the Code

1. Install TypeScript (if not installed)
- npm install -g typescript 
2. ts-node solutions.ts (run this command in terminal)
